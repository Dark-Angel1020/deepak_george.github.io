IOT Enabled Modern Hydroponics System

I worked on an IoT-enabled modern hydroponics project aiming to optimize plant growth. Employing IoT sensors for monitoring temperature, humidity, pH levels, and nutrient concentration, the system ensures ideal conditions. Real-time data acquisition and a centralized monitoring system allow remote access and automated control of environmental factors. This scalable and modular setup enhances crop yield, resource efficiency, and data-driven decision-making, revolutionizing traditional agriculture.

File Compressor using Python

I worked on a Python-based file compressor project. Leveraging various compression algorithms like gzip or zlib, the program reduces file sizes while preserving data integrity. Users can specify compression levels and file types for optimization. Through a simple command-line interface or GUI, the tool offers efficient compression and decompression functionalities, enhancing file management and storage efficiency.

Library Automation with Robots

I worked on a library automation project where I developed a robot equipped with an RFID scanner to streamline book scanning and inventory management processes. The robot autonomously navigates around the library, scanning RFID tags embedded in book covers to identify and record their locations. Utilizing a combination of sensors and algorithms, the robot avoids obstacles and efficiently covers the entire library space. This innovative solution enhances library efficiency by automating tedious tasks, reducing human intervention, and ensuring accurate inventory tracking.

Web Application for Urban Farming Recommendations

Developing a web application for Urban Farming Recommendations powered by Machine Learning (ML) and Deep Learning (DL). This innovative platform harnesses the capabilities of ML and DL algorithms to analyze urban farming data, including soil quality, weather patterns, and crop types, providing personalized recommendations for optimal crop selection, planting techniques, and resource management. By integrating advanced technology with urban agriculture, the application aims to empower individuals and communities to maximize yield and sustainability in limited urban spaces.
